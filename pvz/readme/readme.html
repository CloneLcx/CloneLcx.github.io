游戏简介

植物大战僵尸 (Plants vs. Zombies) 是一款由宝开 (PopCap Games) 开发发布的塔防游戏.

玩家通过种植具有攻击或防御功能的植物来阻止各种僵尸进入房间吃掉脑子.


一代最初发布时只有 PC 和 Mac 版, 理论上支持 Windows 2000 和 OS X 10.4 及以上系统.

后期移植的一代其他平台版本相比电脑版均有不同程度的数据出入和特性裁减, 因此并不推荐.

如果想要在移动平台 (Android/iOS) 上玩, 建议直接下载安装 EA 官方支持的二代和三代.


本站提供的所有资源均为一代 PC 版, 只能顺利运行于 Windows 系统.


版本分类

按照功能特性可分为 "原始版" (2009 年发布) 和 "年度版" (2010 年发布) 两大类.

通常来说, 版本号中涉及 1.0.0.1051 1.2.0.1065 的为原始版, 其他均为年度版 (GOTY Edition).


首次公开发布的 1.0.0.1051 包含两个明显的 bug, 在之后的其他版本中均已得到修复.

- 裂荚射手在前后均有目标时前后均发射两颗豌豆.

- 屋顶场景玉米加农炮射不准, 落点位置向上偏移.


所有版本都有的特性:

- 冒险模式、迷你游戏、解谜模式(我是僵尸/砸罐子)和生存模式(简单/困难/无尽) 四大模式.

- 向日葵奖杯(金向/银向), 禅境花园和智慧树, 植物和僵尸图鉴, 戴夫的道具商店, 通关动画.

年度版主要新特性和变化:

- 成就系统. 大头贴功能. 舞王伴舞贴图替换. 冰车图鉴改动. 冒险模式按钮微调.


这个游戏没有正统的官方中文语言版.

原始版的汉化版由民间团队制作, 翻译文本包含不少的瑕疵, 文件命名不规范, 需谨慎使用.

年度版的汉化版由外包团队制作, 部分名词翻译不准确, 初次发布的版本带广告易崩溃.

使用说明

解压运行

下载压缩包后无需安装, 使用 7-Zip / Bandizip / WinRAR 等软件解压即可, 注意游戏路径不要包含中文.

在文件管理器里导航到解压目录, 鼠标双击 PlantsVsZombies.exe 运行游戏. (建议关闭鼠标设置里的 "提高指针精确度" 选项.)

properties/          配置文件
bass.dll             音乐函数
eula.txt             授权协议
main.pak             贴图素材
PlantsVsZombies.exe  游戏本体
readme.html          自述说明

画质说明

这是一个十多年前的单机小游戏 (≈30MB), 只制作了低分辨率的材质贴图, 没有高清精美的画面效果.

窗口化的像素大小只有固定的 800x600, 高分屏可以临时调低显示器分辨率来玩 (例如 1024x768 1280x720).

由于游戏画面比例 4:3 和主流显示器比例 16:9 不对等, 全屏的时候左右出现黑边属于正常现象.

启动崩溃问题

本游戏在较新的系统上 (win10 v1809+) 存在兼容性问题, 默认以全屏模式启动时会崩溃.

解决方法 1:

添加或者修改以下注册表项目, 让游戏以窗口化模式启动.

(三个项目分别针对: 绝大多数版本, Steam 版, Zoo 日文版)

Windows Registry Editor Version 5.00

[HKEY_CURRENT_USER\SOFTWARE\PopCap\PlantsVsZombies]
"ScreenMode"=dword:00000000

[HKEY_CURRENT_USER\SOFTWARE\SteamPopCap\PlantsVsZombies]
"ScreenMode"=dword:00000000

[HKEY_CURRENT_USER\SOFTWARE\Zoo\Plants vs Zombies]
"ScreenMode"=dword:00000000

也可以直接下载 注册表文件 双击导入. (整合包里已有该文件.)

作用是把游戏设置的 "全屏" 改成 "非全屏"(即窗口化), 导入之后依然不能全屏运行.

解决方法 2:

给系统添加英语语言包并切换为英文输入法, 即可正常全屏.

设置 -> 时间和语言 -> 语言 -> 添加语言 English (United States), 至少勾选 "基本输入" 并安装.

使用快捷键 (按住徽标键并重复按空格键) 或者鼠标点击托盘区域切换到英文输入法 (图标应显示为 ENG).

卡顿问题

如果是笔记本电脑的话连接电源适配器使用, 电源选项调节为 "平衡" 或者 "高性能".

从显卡品牌官网下载更新显卡驱动程序, 从微软网站下载并安装 DirectX 9.0c 运行库. 然后开启游戏内的 "3D Acceleration" 选项.

下载 WineD3D, 解压出 d3d9.dll libwine.dll wined3d.dll 和游戏本体放在同一文件夹下. 然后开启游戏内的 "3D Acceleration" 选项.

Windows 10 系统进设置关闭 "Xbox Game DVR" 功能.

使用窗口化运行而不是全屏运行, 可导入上述注册表文件来更改.

更换用来输出画面的显卡. (双显卡笔记本可尝试连接外屏, 台式将显示器传输线连接到独显接口上.)

关闭其他可能在后台占用资源的软件.
